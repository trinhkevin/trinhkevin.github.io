MY WORK

Hera (Python)

Basketball stastics tracker; created by request for a friend.

SciBot (Python)

Data Science research bot that parses a research paper database and mines for entities, types, collaborators, and associations utilizing Naive Bayes, KMeans, and other methods.

TwinSight (Python/Web/SQL)

Web application that scrapes Twitter for tweets involving Notre Dame football during game days. The web application then uses sentiment analysis and displays tweet density on a world map using Google Maps API.

Brickbreaker (Python)

Multiplayer Brickbreaker game made with the PyGame library played over the network using Twisted.

Pataka App (Android/Java)

Ported iOS app (mostly UI/UX) to Android that detects health issues through speech pathology utilizing the pocketsphinx library. This was done for Undergraduate Research.

Chess (C++)

Worked in a group of four to create a visual Chess simulation using SDL2. Included a recursive AI opponent with three difficulties.

Connect 4 (C)

Created an interactive visual Connect 4 game that utilized X11.
