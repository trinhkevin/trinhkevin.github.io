PORTFOLIO

  • Brickbreaker (Python) - Multiplayer Brickbreaker game made with the PyGame library played over the network using Twisted.
  • Mire (Python) - NFA recognizer/simulator that matched regular expressions. Extended to a finite transducer and then a Turing complete programming language (Turing tarpit).
  • File System (C) - Barebones file system that modifies data on a disk emulator.
  • Virtual Memory (C) - Simulates a virtual memory on the disk and uses varying algorithms to evict data from physical memory.
  • Multithreaded Website Word Parser (C++) - Finds word occurences on websites using a multithreaded producer/consumer queue.
  • Parallel Mandelbrot (C) - Draws a "movie" out of multiple Mandelbrot sets (fractal structure) using multiprocessing and multithreading.
  • Pataka App (Android/Java) - Ported iOS app (mostly UI/UX) to Android that detects health issues through speech pathology utilizing the pocketsphinx library. This was done for Undergraduate Research.
  • Chess (C++) - Worked in a group of four to create a visual Chess simulation uing SDL2. Included a recursive AI opponent with three difficulties.
  • Connect 4 (C) - Created an interactive visual Connect 4 game that utilized X11.
